# Orios Bank Frontend

A modern, elegant, and responsive user interface for the Orios Bank API.

## Features
- **Modern Light Theme**: Clean, elegant light mode design with soft shadows and gradients.
- **Fully Responsive**: Works seamlessly on desktops, tablets, and mobile devices.
- **Smooth Animations**: Floating background blobs, page transitions, hover effects, and interactive elements.
- **Complete Banking Flow**:
    - Secure Login & Registration
    - Dashboard Overview (Total Balance, Accounts)
    - Detailed Account Management
    - Transaction History
    - Fund Transfers
    - Loan Management (Request, Repay, View Status)
- **Admin Panel**:
    - System Overview with Database Statistics
    - Customer Management (View, Delete, Change Password)
    - Account Management (View, Block, Hold, Delete)
    - Loan Management (Approve, Reject, Toggle Auto-Debt)
    - Transaction History Viewer
- **Real-time Notifications**: Instant feedback for all user actions.

## How to Run
1. Ensure the backend Spring Boot application is running on `http://localhost:8080`.
2. Open `index.html` in any modern web browser.
3. You can use a simple static server (like `live-server` or `python -m http.server`) for the best experience.

## Tech Stack
- HTML5
- CSS3 (Grid, Flexbox, Animations, Gradients)
- Vanilla JavaScript
- FontAwesome (Icons)
- Google Fonts (Inter)
